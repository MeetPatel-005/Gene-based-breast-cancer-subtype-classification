    About the Model

    This classifier uses a Stacking Ensemble that combines Random Forest, XGBoost, and LightGBM as base learners, with a meta-learner selected via 5-fold stratified cross-validation.

    Trained on the TCGA-BRCA cohort — gene expression data normalised to TPM (Transcripts Per Million). The model expects Ensembl gene IDs as features and outputs class probabilities across all five subtypes.
